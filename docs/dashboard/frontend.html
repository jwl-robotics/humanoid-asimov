▍ Stage 2 — Front-End Sharpening

essential-matrix refitIRLS on all inliershonest 2 mradgyro-gated, IMU-independent

The VIO's vision measurement is a camera relative-rotation. It sat at ~8 mrad (7.69 median) — the recurring limiter of the whole project (the fusion only won when it over-trusted vision). A diagnosis found the estimator, not the feature tracks, was the bottleneck; a refit sharpened it to ~2 mrad and made the VIO win honest.

The diagnosis — the tracks were fine, the solver wasn't

findEssentialMat + recoverPose returns the 5-point minimal model: RANSAC picks the best 5-point sample, and the many inliers vote for it but never enter the fit. So the rotation error is set by 5 noisy pixels no matter how many features agree. Proof: on clean geometry with 0.4 px synthetic noise the shipped pipeline still produced 6.0 mrad — ~27× the √N correspondence floor of 0.22 mrad; an IRLS refit on the same points gave 1.3 mrad.

  • The long tail was R–t coupling on the coplanar floor (83% of inliers) — twisted-pair branch flips even with perfect correspondences.
  • The ~2 mrad bias was correlated KLT chain drift (template inertia against the loop's rotational flow, invisible to the forward–backward check).
  • Exonerated: the conventions (oracle error 0.00) and rotation-only estimators (Kabsch: 278 mrad — at 0.4–0.9 m walking baselines translation must be modeled as a nuisance, not ignored).
The fix — refit on all inliers, gyro-gated
Left: the minimal-model error (7.69 mrad median, 4.03 bias) vs the IRLS refit (2.06, 0.36) — the refit recovers the √N averaging. Right: at the honest σ_vis = 2 mrad, VIO beats the ESKF on every seed.
  • Huber-IRLS Gauss–Newton refit of (R, t) on all inliers, minimizing the Sampson epipolar error on the SO(3)×S² manifold (translation a modeled nuisance, then discarded).
  • Gyro + neck-FK prior as an accept/reject-only gate (loose 30 mrad): it kills the coplanar branch-flips but never enters the cost — the accepted value is a stationary point of the purely-visual objective, so the measurement stays independent of the IMU the filter already integrates.
  • Anchor age capped at 2 s: a measured negative result — 0.5 s spans pass the money gate but lose in fusion, because below ~1 s the gyro out-informs vision.
  • Textured far walls: the long-anchor workhorse (slow-moving, low-warp, real parallax at 2–3 m baselines). A close-structure variant measurably backfired — occlusion + feature-budget theft.
